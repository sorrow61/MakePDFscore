# 🎵 교회 악보 PDF 생성기 (MakePDFscore)

이미지 형태의 악보 파일들을 번거로운 작업 없이 하나의 **A4 규격 PDF**로 합쳐주는 도구입니다. 
교회 방송팀에서 찬양 악보를 쉽고 빠르게 준비하기 위해 제작되었습니다.

## ✨ 주요 기능
- **폴더별 자동 병합**: `설교전`, `설교후` 폴더의 이미지를 순서대로 합칩니다.
- **A4 최적화**: 모든 이미지를 비율 유지하며 A4 세로 규격에 꽉 차게 배치합니다.
- **강조 문구 자동 삽입**: '설교후' 첫 번째 악보 상단에 **"설교 후 찬양"** 문구(굵은 파란색)를 자동으로 추가합니다.
- **숫자 순 정렬**: 파일명 앞의 숫자를 인식하여 (1, 2, 10...) 정확한 순서로 배치합니다.

## 📂 폴더 구조
```text
.
├── 설교전/          # 설교 전 찬양 악보 이미지 (숫자 순 정렬)
├── 설교후/          # 설교 후 찬양 악보 이미지 (숫자 순 정렬)
├── make_pdf.py     # 메인 파이썬 스크립트
├── PDF만들기.bat    # 더블클릭 실행 파일
└── README.md       # 프로젝트 설명서
```

## 🚀 사용 방법
1. `설교전`, `설교후` 폴더에 각각 악보 이미지 파일들을 넣습니다.
   - 예: `1. 찬양제목.jpg`, `2. 찬양제목.png`
2. **`PDF만들기.bat`** 파일을 더블클릭합니다.
3. 같은 폴더에 생성된 **`악보모음집.pdf`** 파일을 확인합니다.
